📡 Comment utiliser BlueForce ?

  • Mode Basique : Ajoutez des appareils un par un via la popup du navigateur (sécurité obligatoire)
  • Mode Avancé 🔬 : Scannez TOUS les appareils en temps réel avec leur signal ! Cliquez ensuite sur "Connecter" (une popup s'ouvrira quand même pour la sécurité)
  • Connexions multiples : Vous pouvez connecter plusieurs appareils simultanément
  • Nom réel : Le vrai nom s'affiche après connexion (pas juste l'adresse MAC)
  • Historique : Vos appareils sont mémorisés pour un accès rapide
  • iPhone / iPad : Utilisez l'app Bluefy pour accéder au Bluetooth

💡 Le mode avancé nécessite Chrome avec les fonctionnalités expérimentales activées
