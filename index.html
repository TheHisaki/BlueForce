📡 Comment utiliser BlueForce ?

  • Ajouter un appareil : Cliquez sur "Ajouter un appareil Bluetooth" - une popup du navigateur s'ouvrira pour sélectionner l'appareil (sécurité obligatoire)
  • Connexions multiples : Vous pouvez connecter plusieurs appareils simultanément
  • Nom des appareils : Le vrai nom s'affiche après connexion (pas juste l'adresse MAC)
  • Historique : Vos appareils sont mémorisés pour un accès rapide
  • iPhone / iPad : Utilisez l'app Bluefy pour accéder au Bluetooth
